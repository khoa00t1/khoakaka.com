## Built for the whole family

Family Tree (internally **Clan**) is a single Flutter app that runs on
**Android, iOS and the Web**. Pick the device that's already in someone's hand
and they can start adding relatives right away.

Behind the scenes it's powered by **Firebase Authentication, Cloud Firestore
and Firebase Storage**, so changes sync instantly across every member of your
clan.

## Made for real families

* Invite the whole clan with a single share link.
* Capture relationships with the precision genealogists expect, while staying
    friendly enough for kids and grandparents.
* Import existing records from a CSV — no retyping.
* Premium features unlock through a clean in-app subscription.
